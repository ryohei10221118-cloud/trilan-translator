# Trilan Translator - 三語翻譯工具

一個支援中文（繁體/簡體）、英文、韓文三語翻譯的工具，具備辭庫、句庫和分類管理功能。

## 功能特色

- **翻譯功能**：支援繁體中文、簡體中文、英文、韓文互譯
- **辭庫管理**：新增、編輯、刪除單詞翻譯
- **句庫管理**：新增、編輯、刪除句子翻譯
- **分類管理**：為辭庫和句庫建立分類，方便管理

## 使用方式

直接在瀏覽器中開啟 `index.html` 即可使用。

## 技術棧

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage（本地資料存儲）

## 資料存儲

所有資料存儲在瀏覽器的 LocalStorage 中，不需要後端伺服器。
